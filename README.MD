# API of mangas
The API is connected to an DB

## By author **Done**
url = /api/authors
Gets all the mangas of the author and the content (description, review, rating) 
## By serie **Done**
url = /api/series
Gets all the mangas by serie and the content (description, review, rating)
## By chapter **Done**
url = /api/chapters
Gets the content of the chapter (description, review, rating)
## CSV import **In progress**
Create a csv dir in the root of the project, here you can add csv files for each table (auhtor, serie, chapter, serie-author)
